READ ME!!!


How to start and play PyMemory:
1. Click the app to open the game.
2. Press play to choose the preferred grid size.
3. Click the squares on the board to try and match each card pair.
4. You can press pause (Escape key) to re-choose grid size or return to the Main menu.
5. After matching all pairs, press restart to try again.
6. Enjoy and have fun!


What is PyMemory?
PyMemory is a card memorization game wherein players try to guess and match each card pair. Players can choose between the two grid sizes which offers different amounts of cards that the players would try to match and guess. PyMemory helps players sharpen their memory while providing fun and engaging gameplay.
